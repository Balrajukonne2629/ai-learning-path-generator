# AI Learning Path Generator 

## Overview
An AI-powered web application that generates personalized learning roadmaps based on user skills and goals.

## Features
- AI-generated roadmap using Gemini API
- Step-by-step learning plan
- Resource recommendations
- YouTube video suggestions
- Gmail MCP integration for email delivery
- Fallback handling for API failures

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- AI: Gemini API
- MCP: Gmail (Composio)

## How it works
1. User enters skills and goal
2. Backend calls Gemini API
3. Structured roadmap is generated
4. Optional: Sent via email using MCP
